# 1. 产品目标

## 1.1 背景

OpenClaw 是一个 **本地运行 AI Agent 的 runtime**，允许 AI 执行：

* Shell
* 文件系统
* 浏览器
* API 调用
* 自动化任务

目前控制方式：

* Terminal
* Telegram
* Discord

这些方式 **缺少完整 GUI 控制层**。

---

## 1.2 产品目标

开发 **OpenClaw Mobile**：

一个 iOS App，用于：

* 远程控制 Agent
* Chat with Agent
* 管理 Session
* 监控 Agent
* 审批 Tool Execution
* 管理 Automation Tasks

---

## 1.3 设计原则

核心原则：

```
Chat is not the product
Agent control is the product
```

设计重点：

1. 实时性
2. 可观察性
3. 安全性
4. 可扩展性

# 5.1 Authentication

连接 Agent 通过：

```
connection token
```

连接方式：

```
claw://connect?token=xxxxx
```

---

### POST /auth/connect

Request:

```
{
  "token": "xxxxx",
  "device_id": "iphone-123"
}
```

Response:

```
{
  "agent_id": "agent-001",
  "agent_name": "local-agent",
  "model": "claude-3",
  "permissions": ["chat", "tasks"]
}
```
